Conversor de Tempo

Observações:

  1. Esse é um site para estudo.
  2. Para número quebrado como "1,2", use a vírgula, o ponto não irá funcionar.
  3. Alterando o valor (números), clique em converter de novo, para mudar o resultado.
  4. Estamos trabalhando para deixar isso automático. 😉
  5. Valor muito grande, vai ser representado como "3.6e+15", que significa: 3,6 x 10^15 (3600000000000000).
  6. E valor muito pequeno, vai ser representado como "3,6e-15", que significa: 3,6 x 10^-15 (0,0000000000000036).
  7. Lembrando que os valores "3,6", "+15" ou "-15" são EXEMPLOS.
